# Victor Bravo Personal Website

Deployable React/Vite portfolio for MD/PhD application preparation, research, clinical experience, leadership, CV highlights, presentations, and contact.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is generated in `dist/` and can be deployed to Vercel, Netlify, GitHub Pages, or any static hosting provider.
